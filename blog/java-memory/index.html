Java Memory

Klassisches Kartenspiel Memorie, das mit Java für die Schule programmiert wurde

In der Schule hatten wir damit angefangen, ein sehr bekanntes Spiel zu programmieren. Nur leider haben wir es nicht komplett fertig programmiert, daher dachte ich mir: Wieso das Spiel nicht fertig machen?
Gesagt, getan!
Nun ist das Spiel endlich fertig und man kann es direkt über meine Webseite herunterladen. Klickt dazu einfach unten auf Herunterladen und installiert es.

Solltet ihr Probleme beim Installieren oder Starten des Spiels haben, fühlt euch so frei und meldet euch bei mir. Ihr habt einen Fehler entdeckt oder einen Vorschlag für das Spiel? Dann meldet es ganz einfach hier. Das hilft mir, das Spiel zu verbessern.
